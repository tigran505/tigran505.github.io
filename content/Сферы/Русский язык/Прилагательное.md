- Имя прилагательное согласуется с [[Существительное]]:
    - в роде (длинный тюль) *мужской род*
    - в числе (известные люди) *множественное число*
    - в падеже (нет прилежн**ого** ученик**а**) *родительный падеж*
- Имя прилагательное бывает:
     - полными (отвечают на вопрос Какой? Какая? Какое? Какие?) могучий, ленивый, красивый
     - краткими (отвечают на вопрос Каков? Какова? Каково? Каковы?) могуч, ленив, красив
## Степени сравнения
1. Сравнительная простая (красивее)
2. Сравнительная составная (более красивее)
3. Превосходная простая (умнейший)
4. Превосходная составная (этот мальчик более умный)

Имя прилагательное делится на разряды:

| Разряд         | Что обозначает                                                                     | Ключевое отличие                                                                                                                                                  | Примеры                                      |
| -------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------- |
| Качественное   | Качество предмета (величина, возраст, цвет, вес, внешний вид, внутренние качества) | - можно образовать сравнительную степень (более сильный, самый молодой)                                - можно образовать краткую форму (силён, молод, умён, мал) | Сильный, молодой, умный, большой, маленький  |
| Относительное  | Отношение к чему-либо (материалу, месту, времени, лицу, понятию)                   | нельзя образовать сравнительную степень или краткую форму                                                                                                         | деревянный, русский                          |
| Притяжательное | Признак предмета по принадлежности человеку/животному                              | нельзя образовать краткую степень                                                                                                                                 | лисий, олений, кабаний, волчий, мамин, папин |

## 15 задание (отымённые прилагательные)
- Образовано от сущестивительных
	- Лист - лиственный
	- Старина - старинный 
### Общее правило 

| -Н- пишется                                                                                                                                            | -НН- пишется                                                                                                                                                |
| ------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| В суффиксах -ан-/-ян-/-ин-<br>**Примеры:** песчаный, кожаный, серебряный, орлиный, куриный<br>**Исключения:** деревяННый, оловяННый, стекляННый        | В суффиксах -онн-/-енн-<br>**Примеры:** традиционный, естественный, особенный, общественный, многочисленный <br>**Исключение**: ветреный (но безветренный!) |
| Непроизводные (первообразные) прилагательные <br>**Примеры:** пряный, синий, зелёный, багряный, единый, рьяный, пьяный, вороной, румяный, юный, свиной | На стыке основы на -н- и суффикса -н- (стык морфем)<br>**Примеры:** длинный, каменный, барабанный, старинный, холстинный, диковинный                        |
|                                                                                                                                                        | Нужно запомнить<br>**Примеры**: истинный, недюжинный, подлинный, преданный                                                                                  |

> [!warning] **Краткие прилагательные**
> В кратких прилагательных пишется столько же -Н-, сколько и в полных
> **Примеры**: щёки румяные - щёки румяны 
> люди обыкновенные - люди обыкновенны
### Подробно о каждом случае 
#### Одна -Н- 

| Одна -Н- пишется в...                                                                     | Примеры, исключения                                                        |
| ----------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **Суффиксе ИН**: он всегда ударный. Выражает принадлежность к животным, насекомым, птицам | курИНый = курица + ИН                                                      |
| **Суффиксах АН/ЯН:** они обозначают материал                                              | глинЯНый = глина + ЯН<br>**Исключения:** деревяННый, оловяННый, стекляННый |
| **Притяжательных прилагательных** (отвечают на вопрос "чей?")                             | фазаНий, олеНий (Н является частью корне)                                  |
| Названиях растений при наличии суффикса ОВ                                                | апельсиНОВый, жасмиНОВый                                                   |

#### Две -НН-

| Две НН пишутся в...                                                                 | Примеры, исключения                                                                                                   |
| ----------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| **Суффиксе ОНН:** всегда ударный; часто иноязычного происхождения                   | редакциОННый = РЕДАКЦИЯ + ОНН                                                                                         |
| **Суффиксе ЕНН**: обычно от основы на согласную букву                               | искусствЕННый = ИСКУССТВ + ЕНН<br>невежествЕННый = НЕВЕЖЕСТВ + ЕНН<br>**Исключение**: ветреный (но безветренный!)     |
| В прилагательных, образованных от существительных на -МЯ                            | безымЯННый (от имя)<br>времЕННый (от время)                                                                           |
| В прилагательных на стыке морфем (корня и суффикса)                                 | длиННый (длина + Н)<br>истиННый (истина + Н)<br>камеННый (камень + Н)<br>ценный (цена + Н)<br>недюжинный (дюжина + Н) |
| В прилагательных, обозначающих названия растений без суффикса ОВ (основа + Н)       | апельсиННый, жасмиННый                                                                                                |
| В прилагательных с беглым е между НН в форме краткого прилагательного мужского рода | надменный - надменЕн<br>постояННый - постоянЕН<br>обыкновенный - обыкновенЕн                                          |
