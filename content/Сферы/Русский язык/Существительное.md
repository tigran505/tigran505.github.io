## Категорию рода
- Женский род (ученица)
- Мужской род (ученик, тюль)
- Средний род (солнце, мороженое)
 - Общий род (гуляка, писака, умница). Может быть и к М, и к Ж
- Нет рода у слов, употребляемых только во множественном числе (очки, джинсы, ножницы)
## Число
- Единственное число (ученик)
- Множественное число (ученики)
## Падеж *(Иван родил девчонку, велел тащить пелёнку)*
- Именительный падеж (Кто? Что?) гуляка
- Родительный падеж (нет Кого? Чего?) гуляки
- Дательный падеж (рад Кому? Чему?) гуляке
    - Винительный падеж (вижу Кого? Что?) гуляку
    - Творительный падеж (любуюсь Кем? Чем?) гулякой
    - Предложный падеж (говорю О ком? О чём?) о гуляке
- Также имена существительные делятся на 3 склонения (см. [[Склонения существительных]])

## 15 задание
### -Н-
#### Непроизводное прилагательное с Н
- Юный → юность 
- Пряный → пряность
- Багряный → багряница 
#### От прилагательного с основой на Н 
##### Первый случай
- @ Гость + -ИН- → гостИНый / гостИНая (для гостей)
- Гостиный + -ИЦ- → гостиница 
##### Второй случай 
- @ Песок + -АН- → песчАНый 
- Песчаный + -ИК- → песчаник 
##### Третий случай
- Торф + -ЯН- → торфЯНой 
- Торфяной + -ИК- → торфяник 
#### От отглагольного прилагательного с Н 
- **Варёный** + -ИК- → **вареник** 
- **Копчёный** + -ОСТЬ- → **копчёность**
- **Масленый** + -ИЦ- → **Масленица**
- **Мудреный** + -ОСТЬ- → **мудреность** 
- **Мученый** + -ИК- → **мученик**
- **Пудрёный** + -ИЦ- → **пудреница**
- **Путаный** + -ИЦ- → **путаница** 
- **Смышлёный** + -ОСТЬ- → **смышлёность**
- **Труженый** + -ИК- → **труженик**
### -НН-
#### От существительных с основной на Н + Н 
- Дружина → дружинник 
- Конь → конник → конница 
- Осина → осинник 
- Измена → изменчик 
#### От прилагательных с НН

- Образованный + -ОСТЬ- → образованность
- Торжественный + -ОСТЬ- → торжественность
