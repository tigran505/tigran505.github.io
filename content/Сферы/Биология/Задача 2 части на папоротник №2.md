Какой хромосомный набор характерен для заростка и взрослого растения папоротника? Объясните, из каких исходных клеток и в результате какого деления они образуются.
## Решение 
1. Для заростка папоротника характерен гаплоидный n набор хромосом. Заросток папоротника образуется из гаплоидной n споры путём митоза. 
2. Для взрослого растения папоротника характерен диплоидный 2n набор хромосом. Взрослое растение папоротника образуется из диплоидной 2n зиготы путём митоза
