Какой хромосомный набор характерен для клеток ризоидов [[Отдел Бурые водоросли|ламинарии]], закрепляющих листовую пластинку, а также для спор и клеток женского гаметофита? В результате какого деления и из каких клеток образуются эти клетки? Свой ответ поясните.
## Решение 
1. Для клеток ризоидов ламинарии, закрепляющих листовую пластину, характерен диплоидный 2n набор хромосом. Взрослая особь, представленная спорофитом, образуется из 2n зиготы путём митоза  
2. Для спор характерен гаплоидный n набор хромосом. Споры образуются путём мейоза клеток спорангия, представленного клетками на поверхности листовидных талломов
3. Для клеток женского гаметофита характерен гаплоидный n набор хромосом. Примерно половина спор развивается в клетки женского гаметофита (это происходит путём митоза)
