[[Вода]] поступает в растения по осмотическому градиенту, который создается в [[Корень растения|корнях]]. При этом градиент формируется в основном за счет ионов [[Калий|K]]<sup>+</sup> и [[Хлор|Cl]]<sup>-</sup>, концентрация которых в центральном цилиндре оказывается больше, чем в периферических слоях корня. Существует два вида транспорта ионов через мембрану. Назовите их. Чем они различаются? Почему при обработке растений [[Ингибиторы|ингибитором]] АТФ-синтетазы транспорт воды в корне прекращается? Как изменится скорость поступления воды в корень пшеницы, пересаженной в [[Почва|почву]] с большей концентрацией солей? Ответ поясните.
## Решение 
1. [[Активный мембранный транспорт]] против градиента концентрации, с затратой энергии ([[АТФ]]), происходит при участии мембранных насосов (белков-переносчиков). У растений активный транспорт используется для поглощения минеральных веществ
2. [[Пассивный мембранный транспорт]] по градиенту концентрации, без затрат энергии (АТФ). *У растений пассивный транспорт используется для поглощения воды*. Пассивный транспорт ионов происходит через белковые каналы  
3. При блокировке [[Ферменты|фермента]] АТФ-синтетазы снижается количество синтезируемой АТФ. Без АТФ не работают белковые насосы => при уменьшении количества АТФ нарушается активный транспорт ионов
4. Снижается количество ионов в корне => снижается [[Осмотическое давление|осмотическое давление]] 
5. Скорость поступления воды снизится. Концентрация солей в засоленной почве (почвенном растворе) выше, чем в клетках корня => вода не может поступать в клетки корня
